# DalaGET-ver.1
A locator app mainly focused on the town of dalaguete


# Main Functions: 
- a map system of the town of dalaguete
- a search feature
- tracking system
  
  

# Side functions:
- chatbot (using Ollama platform)
- search filter
- Local businesses can register so they can show on map or in search
- Review & ratings
- Emergency button
